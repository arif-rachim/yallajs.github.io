To start building yallajs, all you need todo is import https://unpkg.com/yallajs in document.head.

Then you need to get the html Tagged Template Literal from Context object, by using let {html} = new Context().

After that you can use render function to render html to document node. In this example we are using document.body

Template Literal

Template literals are string literals allowing embedded expressions. You can use multi-line strings and string interpolation features with them.

Template literals are enclosed by the back-tick (` `) (grave accent) character instead of double or single quotes. 

Template literals can contain placeholders. These are indicated by the dollar sign and curly braces (${expression}). The expressions in the placeholders and the text between them get passed to an html function.
