alla

10kb High Performance Non VirtualDOM engine + SPA Inclusive

Javascript framework for building webapp swiftly

Friend of Developer and Designer

Unlike other javascript libraries, YallaJS simply uses the native web language, "Javascript". By using Javascript, you do not need the compiler to "transpile" your application. This means that your development will be much faster, you can debugging natively through the browser, and you can use DevTools source mapper that you can not use if you use JSX or TypeScript. sweeet!

Templating with JSX makes it easy for Web Developer to make sure the code that is created does not break. For this yalla js use ES6 String Literals, so your templating will be much easier and you get autocomplete feature from your favorite javascript editor.

High Performance Feather Lite

YallaJS is very light, total size no more than 10kb. With a very small size, the bandwidth used is very cheap and indirectly much faster.

YallaJS does not use virtual DOM. To produce a high performance, Yallajs cached the parsed string literal into an HtmlTemplate. Then yallajs map the position of the dynamic parameter in HtmlTemplate into a Placeholder. So if there is a change of parameter value, yallajs simply update the effected placeholder. In this way the application is more efficient in memory usage, and the script computation usage is much lower.

Trully Portable & Embeddable

You can create an application using yallajs as hybrid or single page application. By using hybrid, you can embed yallajs component into your larger web application. If you want to feel the ability of single page application, you can use yalla spa to get routing facility and integration with web component.

Yalla Js is compatible with all browsers that support ES5. Yalla Js runs well in Internet Explorer 11, iOS 9, Android 5, and other modern browsers.

Browserstack is the platform we use for crossbrowser testing and we thank them for their support.
