YallaJS ES6 Template Literal Templating Library

YallaJS is high performance, small size footprint and light weight ES6 Templating Engine library.

Getting Started

This guide walks through building a simple Hello World.

  1. Import yallajs in your index.html
  2. Create main javascript file (main.js).
  3. Import main.js into index.html
  4. Open browser to see the changes
