Components
Component help you make your UI becomes independent, reusable and can be tested individually.
In yallajs you build your components using simple HTML. You can declare your stylesheeet in the component and the stylesheet will be isolated in the component.

Let's create our first component "hello-world".

  • Create a file named hello-world.html in the base folder
  • Now in the hello-world.html you can start writing html
            hello-world.html
<div>Hello World</div>
  • Now from browser you can start calling your components by using HTML5 hashbang
  • http://localhost:8080/index.html#hello-world

Lets give properties to component

  • We want to use name properties in the component
  • We can use curly braces to input expression in the component
                hello-world.html
<div>Hello World {$props.name}</div>
  • Now from browser you can start passing the properties to the component
  • http://localhost:8080/index.html#hello-world:name=Jonny Papa
  • Your component will render Hello World Jonny Papa
  • Properties passed from browser is colon separated.
                hello-world.html
<div>Hello World {$props.firstName} {$props.lastName}</div>
  • Now from browser you can start passing the properties to the component
  • http://localhost:8080/index.html#hello-world:firstName=Jonny Jonny:lastName=Yes Papa
  • Your component will render Hello World Jonny Jonny Yes Papa

Lets put some styles to the component

  • We want to style the component
  • We can use style tag in inside the component
                hello-world.html
<div>
  <style>
      root{
         font-size : 1em;
      }
      .label {
         background-color : #CCCCCC;
      }
  </style>
   Hello World <label class="label">{$props.firstName} {$props.lastName}</label>
</div>
  • root selector will be apply to the component root (in this example it is a DIV tag)
  • We can define all the rules in style tag and the rules will be isolated in this component
