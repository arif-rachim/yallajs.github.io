Composition
Composition is a way to incorporate elements from a parent component into sub components. In our previous example the parent component user-card is hello-world.
Now if we want to put the composition into a user-card element we can use the attribute $ children.

Let's get back to "user-card" component.

  • in user-card component, lets create a new div where we will put the composition from the parent using $children.
            user-card.html
<div>
    <h3>{$props.firstName} {$props.lastName}</h3>
    {$props.email}
   <div $children>
   </div>
</div>
  • What happens is, we make a div tag, and defining attribute $children.
  • The DIV tag will be a container of a given element composition from the parent.

Now in the "hello-world" component

  • We declare the composition elements for user-card.
            hello-world.html
<div>
    <inject name="thisIsUserCard" value="docs-sample/user-card"/>
    Hello World
    <this-is-user-card first-name="{$props.firstName}"
                       last-name="{$props.lastName}"
                       email="{$props.email}">
        <div>
            <button>Hello World Button</button>
        </div>
    </this-is-user-card>
</div>
  • Now from browser you can call
  • http://localhost:8080/index.html#hello-world:firstName=Jonny:lastName=Papa:email=JonnyPapa@gmail.com
  • You can see now the user-card renders the Hello World Button inside the component.
