Foreach
We can use the foreach directive to render a list of items based on an array. The foreach directive requires a special syntax in the form of item in items, where items is the source data array and item is an alias for the array element being iterated on.

Lets update the table in 'user-list'

  • foreach syntax should use following rules
  • foreach="item,index in arrayToIterate"
    •                 contact-manager/user-list.html
<div>
   <var name="contacts" value="{actions.getContacts()}" />
    <table>
        <tr>
            <th>First Name</th>
            <th>Last Name</th>
            <th>Email</th>
            <th>Phone</th>
        </tr>
        <tr foreach="contact in contacts">
            <td>{contact.firstName}</td>
            <td>{contact.lastName}</td>
            <td>{contact.emailName}</td>
            <td>{contact.phoneNo}</td>
        </tr>
    </table>
</div>
    • Now if we call this component from browser it will generate the contact based on the foreach statement
