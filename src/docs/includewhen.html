Conditional Rendering
Conditional rendering provides the option to not render at all in the browser dom. We can use attribute $includewhen with expression to check. The expression should return javascript boolean true or false. Conditional rendering is not like css attribute display: none,it will not going to make dom in the browser at all instead hiding it , hence it will reduce the memory usage.

Lets update the table in 'user-list'

  • To check Conditional Rendering we can use attribute $includewhen just like following example
                contact-manager/user-list.html
<div>
    <var name="contacts" value="{actions.getContacts()}" />
    <table $includewhen="{contacts && contacts.length > 0}">
        <tr>
            <th>First Name</th>
            <th>Last Name</th>
            <th>Email</th>
            <th>Phone</th>
        </tr>
        <tr foreach="contact in contacts">
            <td>{contact.firstName}</td>
            <td>{contact.lastName}</td>
            <td>{contact.emailName}</td>
            <td>{contact.phoneNo}</td>
        </tr>
    </table>
</div>
  • Above code means we will render the table if we have contacts data
