Dependency Injection
Dependency injection allows you to use components from outside into your component. To use dependency injection you can use the inject tag.

Let's create our second component "user-card".

  • Create a file named user-card.html in the base folder
  • Now in the user-card.html you can start writing html
            user-card.html
<div>
    <h3>{$props.firstName} {$props.lastName}</h3>
    {$props.email}
</div>
  • Now from browser you can start calling your components by using HTML5 hashbang
  • http://localhost:8080/index.html#user-card:firstName=Johny:lastName=Papa:email=JohnnyPapa@gmail.com

Let's go back to hello-world component and use "user-card".

  • Go back to hello-world.html
  • Now in the hello-world.html you can inject user-card
                hello-world.html
<div>
  <inject name="thisIsUserCard" value="user-card"/>
   Hello World
   <this-is-user-card></this-is-user-card>
</div>
  • When you use inject, the name is the component name you can use. You can declare name using camelCase, and when you want to use the component you need to use hyphen-case
  • Now you can pass properties to this-is-user-card.html by using hyphen-case
                hello-world.html
<div>
   <inject name="thisIsUserCard" value="user-card"/>
   Hello World
   <this-is-user-card first-name="{$props.firstName}"
                         last-name="{$props.lastName}"
                         email="{$props.email}">
   </this-is-user-card>
</div>
  • Now from browser you can start passing the properties to the component
  • http://localhost:8080/index.html#hello-world:firstName=Jonny:lastName=Papa:email=Jonny.papa@gmail.com
  • Your component will render Hello World Jonny Papa Jonny.papa@gmail.com
