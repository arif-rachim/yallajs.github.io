Routing
Routing is the mechanism by which we can call our components directly from the address bar. You can call your components using the hash sign # in the browser address bar, and sub-components delimeted by using the slash sign.
Here is the routing rules in yallajs.
  • You can call components that you put inside package (directory) by using the dot operator '.'. For example, if you put the components in the folder packageA/packageB/my-components.html. To call my-components.html from the browser you can do it like this 'index.html#packageA.packageB.my-components'.
  • You can call subview using slash operator '/'. For example, you can call the following in browser index.html#contact-manager.index/contact-manager.detail:id=jonny. This means that we render index.html component that is inside contact-manager folder,and the sub-components in the folder detail.html contact-manager as well.
  • To show subview in components we can use the tag 'router-view'

Let's create a new component 'user-list'

  • We will save user-list.html component in contact-manager folder.
  • Then in user-list.html we can write the following html
                contact-manager/user-list.html
<div>
    <table>
        <tr>
            <th>First Name</th>
            <th>Last Name</th>
            <th>Email</th>
            <th>Phone</th>
        </tr>
        <tr>
            <td>John</td>
            <td>Rohn</td>
            <td>john.rohn@gmail.com</td>
            <td>06092982</td>
        </tr>
    </table>
</div>
  • We can call the user-list component from the browser address bar with following address
  • index.html#contact-manager.user-list

Let us update hello-world.html

  • In the hello-world components now we can add a router-view tag.
                hello-world.html
<div>
    <inject name="thisIsUserCard" value="docs-sample/user-card"/>
    Hello World
    <router-view></router-view>
    <this-is-user-card first-name="{$props.firstName}"
                       last-name="{$props.lastName}"
                       email="{$props.email}">
        <div>
            <button>Hello World Button</button>
        </div>
    </this-is-user-card>
</div>
  • Now we can see that there is a router-view in the hello-world components.
  • Now from browser you can start opening the component and subcomponent with following address
  • http://localhost:8080/index.html#hello-world/contact-manager.user-list
  • However above component will not render properly because the hello-world component does not receive complete props
  • To pass the properties to hello world component we can put full complete param as following
  • http://localhost:8080/index.html#hello-world:firstName=Jonny:lastName=Papa:email=Jonny.papa@gmail.com/contact-manager.user-list
