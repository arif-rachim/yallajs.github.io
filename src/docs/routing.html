Routing
Routing adalah mekanisme dimana kita bisa memanfaatkan browser history untuk merender komponent kita. Anda bisa memanggil komponen anda menggunakan hash sign # di browser address bar.
Anda dapat memanggil komponen yang anda simpan didalam package (directory) dengan menggunkan operator dot (.). Contohnya apabila anda menaruh komponen di folder packageA/packageB/my-components.html. Maka untuk memanggil komponen my-components dari browser anda dapat melakukannya seperti ini '#packageA.packageB.my-components' .
Now if we want to put the composition into a user-card element we can use the attribute $ children.
