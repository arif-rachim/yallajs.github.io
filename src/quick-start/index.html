Selamat datang di Yallajs
This quick start guide will take you through createding a Todo app using Yallajs and briefly explain its main concepts. We assume you are familiar with Javascript, Html, and CSS.

<html lang="en">
<head>
    <meta charset="UTF-8">
    <script src="https://yallajs.github.io/yalla.js" data-main="app" data-base="src"></script>
</head>
<body>
</body>
</html>
