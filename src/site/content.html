Just one library

Build HTML5 applications require a lot of libraries. Some but not all including library to manage modules, library for building components, library for routing, library to manage the state. With Yallajs you no longer need these separate libraries.

Fast and Lightweight

Although it is equipped with various features and libraries, yallajs weight is less than 20kb.

Focus on applications built

Yallajs help you to start building HTML 5 applications quickly and easily. You simply add yallajs script to index html and then you can start building your application.

Enforce clean code

Yallajs help you to build applications with clean code by avoiding the javascript function in html, or vice versa. Yallajs developer requires them to follow the rules, yallajs will not process javascript function in html.

Simple and easy to learn

Yallajs doesn't require you to understand new concepts, if you already understand html, css and javascript, you will easily be able to start building applications using yallajs.

Stateless functional component

What happens behind is the transformation html component into a javascript stateless functional component. Stateless functional component is very light because of the absence of state and component lifecycle.

Unidirectional State

Yallajs equipped with state container that takes the concept of Redux. Data processed unidirectional, so that easy for developers to predict or to test the state of the application.

Incremental DOM

Yallajs using Incremental dom in the process of updating the DOM. By using incremental DOM, Yallajs still able to produce applications with good performance, without compromising memory usage.

Component Routing

Components built on yallajs can be called directly from the browser address bar. Thus, the components can be serves as View (if called directly from the address bar) or as a sub component of Composite.
