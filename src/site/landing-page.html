Just one library

Build HTML5 applications require a lot of libraries. Some but not all including library to manage modules, library for building components, library for routing, library to manage the state. With Yallajs your application no longer need these separate libraries.

Fast and Lightweight

Although it is equipped with various features and libraries, yallajs is less than 25kb.

Focus on applications built

Yallajs help you to start building HTML 5 applications quickly and easily. Simply add yallajs script to index html and then you can start building your HTML 5 application.

Enforce clean code

Yallajs help you to build applications with clean code by avoiding the javascript function in html, and vice versa. Yallajs requires application developer to follow the rules, yallajs will not process javascript function in html.

Simple and easy to learn

Yallajs doesn't require you to understand new concepts, if you already understand html, css and javascript, you will easily be able to start building applications using yallajs.

Stateless functional component

What happens behind is the transformation html component into a javascript stateless functional component. Stateless functional component is a very light component because the absence of state and component lifecycle.

Unidirectional State

Yallajs equipped with state container that takes the concept of Redux. Data processed unidirectional way, so that easy for developers to predict or to test the state of the application.

Incremental DOM

Yallajs using Incremental dom in the process of updating the DOM. By using incremental DOM, Yallajs still able to produce applications with good performance, without compromising memory usage.

Component Routing

Components built on yallajs can be called directly from the browser address bar. Thus, the components can be serves as View (if it called directly from the address bar) or as Composite component.
